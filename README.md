## Teck stack
- nx
- React
- TailwindCSS
- RecoilJS
- TypeScript

## Getting started
To get a local copy up and running, follow these steps.
run:

- `git clone https://github.com/Tshobohwa/frontend-challenge.git`
- `cd frontend-challenge`
- `npm install`
- `npx nx run todo:dev`
